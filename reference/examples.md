## HOW TO ADD NEW EXAMPLES

**When to add:**
→ Any post that got above-average engagement (save, comment, share spike)
→ Any post that felt most like your voice — even if engagement was average
→ Any Claude output you approved with minimal edits
→ Any hook or CTA you want to repeat or build on

**How to add:**
1. Copy the template for the relevant section
2. Paste the content exactly as posted (including hashtags for Instagram)
3. Add the pillar tag and format
4. Add engagement signal if available
5. Write the WHY IT WORKS annotation — this is non-optional

**The annotation rule:**
A bare example without a WHY IT WORKS note is less than half useful.
The annotation tells Claude what to replicate, not just what to mimic.
Even one line is enough: "The hook works because it names a specific
cost before explaining the concept."

---

## CALIBRATION INSTRUCTIONS FOR CLAUDE

When this file is loaded, before drafting any content:

1. Read all completed examples in the relevant section (match by
   platform and pillar first, then expand to others)
2. Identify the voice patterns present: sentence rhythm, specificity
   level, emotional register, CTA style
3. Note any annotations that flag what worked and why
4. Calibrate the new draft to match — not to copy, but to feel like
   the same writer wrote it
5. If the examples section for a content type is empty: default to
   voice-rules.md and brand-guidelines.md voice samples until
   real examples are added

**The goal is not imitation. It is calibration.**
New content should feel like a natural continuation of the library —
same voice, new idea.
